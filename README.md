# Firefighter Linux

Firefighter Linux is an offline-first disaster response Linux distribution based on Xubuntu 24.04 LTS.

## Goals

- Offline communication
- Incident reporting
- Local SQLite database
- Offline maps
- LoRa / Meshtastic preparation
- Lightweight deployment
- Real-device compatibility

## Features

- Custom branding
- Plymouth boot screen
- GRUB customization
- Welcome screen
- JSON export system
- Offline reporting
- Calamares installer

## Architecture

Ubuntu Host
↓
VirtualBox
↓
Firefighter Linux DEV VM
↓
ISO Production

## Version

Current version: V0.1 DEV
