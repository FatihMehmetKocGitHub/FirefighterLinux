# Firefighter Linux Offline Maps

Bu klasör afet ve internet kesintisi durumları için ayrılmıştır.

## Planlanan Sistem

- OpenStreetMap tile cache
- Türkiye çevrimdışı harita paketleri
- GPX / GeoJSON destekleri
- Meshtastic konum entegrasyonu
- Yerel Leaflet tile server

## Klasörler

- tiles/ → offline harita tile dosyaları
- cache/ → geçici tile cache
- turkiye/ → Türkiye offline harita paketleri

## Gelecek V1.1

- Offline routing
- Afet bölgeleri
- Toplanma alanları
- Yerel mesh konum paylaşımı
